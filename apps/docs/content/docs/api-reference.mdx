# API Reference

Complete API documentation for all Tagaroa backend services.

## Base URLs

| Service | Base URL | Description |
|---------|----------|-------------|
| Auth | `http://localhost:8080/api/auth` | Authentication & authorization |
| Finance | `http://localhost:8080/api/finance` | Financial data management |
| Investment | `http://localhost:8080/api/investment` | Investment tracking |
| Storage | `http://localhost:8080/api/storage` | File storage & management |
