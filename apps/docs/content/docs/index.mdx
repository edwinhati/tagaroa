# Welcome to Tagaroa

Tagaroa is a comprehensive full-stack platform built as a Turborepo monorepo, featuring multiple applications, microservices, and shared packages for building modern web applications.

## Architecture Overview

The platform consists of:

- **Frontend Applications**: Next.js 16 apps for web, auth, admin, finance, and docs
- **Backend Services**: Microservices built with TypeScript (Hono/Bun), Go, and Rust
- **Shared Packages**: Reusable UI components, utilities, and configurations
- **Infrastructure**: Docker, Kubernetes, Traefik, and Flux for deployment

## Quick Start

```bash
# Install dependencies
bun install

# Setup environment files
make env-setup

# Start infrastructure services
make docker-up

# Initialize databases
make db-init

# Run all apps in development
bun run dev
```

## Project Structure

```
tagaroa/
├── apps/              # Frontend applications
│   ├── web/          # Main landing page
│   ├── auth/         # Authentication app
│   ├── admin/        # Admin dashboard
│   ├── finance/      # Finance management
│   └── docs/         # Documentation site
├── servers/          # Backend microservices
│   ├── auth/         # Auth service (TypeScript/Hono/Bun)
│   ├── finance/      # Finance service (Go)
│   ├── investment/   # Investment service (Rust)
│   └── storage/      # Storage service (TypeScript/Hono/Bun)
├── packages/         # Shared packages
│   ├── common/       # Common utilities and hooks
│   ├── ui/           # Shared UI components
│   ├── shared/       # Shared code (Go, Hono, Rust, Proto)
│   ├── posthog-config/  # PostHog configuration
│   ├── sentry-config/   # Sentry configuration
│   └── typescript-config/ # TypeScript configurations
```

## Key Features

- **Monorepo Architecture**: Turborepo for efficient builds and caching
- **Multi-Language**: TypeScript/Bun, Go, and Rust microservices
- **Modern Stack**: Next.js 16, React 19, Tailwind CSS 4
- **Authentication**: Better Auth with OAuth support
- **Analytics**: PostHog integration
- **Error Tracking**: Sentry integration
- **Database**: PostgreSQL with Drizzle ORM (TypeScript) and sqlx (Rust)
- **Message Queue**: Kafka for event-driven architecture
- **Storage**: MinIO for object storage
- **API Gateway**: Traefik for routing and load balancing
- **Automation**: n8n for workflow automation
- **Package Manager**: Bun for fast dependency management
