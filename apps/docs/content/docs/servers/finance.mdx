# Finance Server

Financial data management service built with Go, handling accounts, budgets, and transactions.

## Overview

- **Port**: 8082
- **Technology**: Go 1.24
- **Database**: PostgreSQL
- **Purpose**: Financial account and budget management

## Features

- Account management (CRUD)
- Budget tracking and management
- Transaction processing
- Budget history and analytics
- OIDC authentication
- Kafka event publishing
- RESTful API
- Comprehensive testing

## Data Models

### Account

```go
// internal/model/account.go
type Account struct {
    ID          string    `json:"id"`
    UserID      string    `json:"userId"`
    Name        string    `json:"name"`
    Type        string    `json:"type"` // checking, savings, credit, investment
    Balance     float64   `json:"balance"`
    Currency    string    `json:"currency"`
    CreatedAt   time.Time `json:"createdAt"`
    UpdatedAt   time.Time `json:"updatedAt"`
}

type CreateAccountRequest struct {
    Name     string  `json:"name" validate:"required"`
    Type     string  `json:"type" validate:"required,oneof=checking savings credit investment"`
    Balance  float64 `json:"balance"`
    Currency string  `json:"currency" validate:"required,len=3"`
}
```

### Budget

```go
// internal/model/budget.go
type Budget struct {
    ID          string    `json:"id"`
    UserID      string    `json:"userId"`
    Name        string    `json:"name"`
    Amount      float64   `json:"amount"`
    Spent       float64   `json:"spent"`
    Category    string    `json:"category"`
    Period      string    `json:"period"` // monthly, yearly, custom
    StartDate   time.Time `json:"startDate"`
    EndDate     time.Time `json:"endDate"`

    Name      string    `json:"name" validate:"required"`

    Period    string    `json:"period" validate:"required,oneof=monthly yearly custom"`
    StartDate time.Time `json:"startDate" validate:"required"`
    EndDate   time.Time `json:"endDate" validate:"required,gtfield=StartDate"`
}
```

```
     ↓
  Middleware
     ↓

```

### Repository Pattern

```go
// internal/repository/account_repository.go
type AccountRepository interface {
    Create(ctx context.Context, account *Account) error
    GetByID(ctx context.Context, id string) (*Account, error)
    GetByUserID(ctx context.Context, userID string) ([]*Account, error)
    Update(ctx context.Context, account *Account) error
    Delete(ctx context.Context, id string) error
}
```

### Service Layer

```go
// internal/service/account_service.go
type AccountService interface {
    CreateAccount(ctx context.Context, req *CreateAccountRequest) (*Account, error)
    GetAccount(ctx context.Context, id string) (*Account, error)
    ListAccounts(ctx context.Context, userID string) ([]*Account, error)
    UpdateAccount(ctx context.Context, id string, req *UpdateAccountRequest) (*Account, error)
    DeleteAccount(ctx context.Context, id string) error
}
```

## Authentication

### OIDC Integration

```go
// Middleware validates JWT tokens from auth server
func AuthMiddleware(oidcConfig *OIDCConfig) gin.HandlerFunc {
    return func(c *gin.Context) {
        token := c.GetHeader("Authorization")
        if token == "" {
            c.AbortWithStatusJSON(401, gin.H{"error": "unauthorized"})
            return
        }
        
        // Validate token with OIDC provider
        claims, err := validateToken(token, oidcConfig)
        if err != nil {
            c.AbortWithStatusJSON(401, gin.H{"error": "invalid token"})
            return
        }
        
        c.Set("userID", claims.Subject)
        c.Next()
    }
}
```

## Event Publishing

### Kafka Integration

```go
// Publish account created event
func (s *accountService) CreateAccount(ctx context.Context, req *CreateAccountRequest) (*Account, error) {
    account, err := s.repo.Create(ctx, req)
    if err != nil {
        return nil, err
    }
    
    // Publish event
    event := AccountCreatedEvent{
        AccountID: account.ID,
        UserID:    account.UserID,
        Type:      account.Type,
        Timestamp: time.Now(),
    }
    
    s.kafka.Publish("account.created", event)
    
    return account, nil
}
```

## Testing

### Unit Tests

```go
func TestAccountService_CreateAccount(t *testing.T) {
    repo := &mockAccountRepository{}
    service := NewAccountService(repo)
    
    req := &CreateAccountRequest{
        Name:     "Checking",
        Type:     "checking",
        Balance:  1000.00,
        Currency: "USD",
    }
    
    account, err := service.CreateAccount(context.Background(), req)
    
    assert.NoError(t, err)
    assert.NotEmpty(t, account.ID)
    assert.Equal(t, req.Name, account.Name)
}
```
