# Finance App

Comprehensive personal finance management application for tracking accounts, budgets, and transactions.

## Overview

- **Port**: 3004
- **Framework**: Next.js 16 with App Router
- **Purpose**: Financial tracking, budgeting, account management

## Features

- Account management (bank accounts, credit cards, investments)
- Budget creation and tracking
- Transaction management
- Budget history and analytics
- Data tables with advanced filtering
- Real-time budget calculations
- Date range selection
- Number formatting for currency

## Data Management

### State Management
Uses Zustand for global state:

```typescript
// Budget store
const useBudgetStore = create((set) => ({
  budgets: [],
  addBudget: (budget) => set((state) => ({
    budgets: [...state.budgets, budget]
  })),
  updateBudget: (id, updates) => set((state) => ({
    budgets: state.budgets.map(b => 
      b.id === id ? { ...b, ...updates } : b
    )
  }))
}));
```

### API Integration
Uses TanStack Query for data fetching:

```typescript
// Fetch accounts
const { data: accounts } = useQuery({
  queryKey: ['accounts'],
  queryFn: () => fetch('/api/finance/accounts').then(r => r.json())
});
```
