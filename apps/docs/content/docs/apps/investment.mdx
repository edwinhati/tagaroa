# Investment App

> ⚠️ **In Development**: This application is currently under active development.

Investment portfolio management application for tracking portfolios, assets, and performance analytics.

## Overview

- **Port**: 3005
- **Framework**: Next.js 16 with App Router
- **Purpose**: Investment tracking, portfolio management, performance analytics

## Planned Features

- Portfolio management
- Investment tracking
- Performance analytics
- Asset allocation visualization
- Historical data and trends
- Real-time market data integration

## Project Structure

## Development

```bash
# Run development server
cd apps/investment
bun run dev

# Or from root
turbo dev --filter=investment

# Build for production
bun run build
```

## Environment Variables

```bash
# apps/investment/.env
NEXT_PUBLIC_API_URL=http://localhost:8080
NEXT_PUBLIC_POSTHOG_KEY=your-posthog-key
NEXT_PUBLIC_POSTHOG_HOST=https://app.posthog.com
```
