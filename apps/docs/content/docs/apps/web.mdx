# Web App

The main landing page and public-facing website for the Tagaroa platform.

## Overview

- **Port**: 3000
- **Framework**: Next.js 16 with App Router
- **Purpose**: Public landing page, marketing site, and entry point

## Features

- Modern landing page with hero section
- Responsive header and footer
- Client-side interactivity
- Integration with authentication flow
- Analytics tracking with PostHog

## Project Structure

```
apps/web/
├── src/
│   ├── app/
│   │   ├── layout.tsx      # Root layout
│   │   └── page.tsx        # Home page
│   └── components/
│       ├── client-home-page.tsx
│       ├── header-section.tsx
│       ├── hero-section.tsx
│       └── footer-section.tsx
├── .env.example
├── Dockerfile
├── next.config.ts
└── package.json
```

## Components

### Header Section
Navigation header with branding and menu items.

### Hero Section
Main hero section with call-to-action buttons.

### Footer Section
Footer with links and company information.

### Client Home Page
Client-side rendered home page component.

## Development

```bash
# Run development server
cd apps/web
bun run dev

# Or from root
turbo dev --filter=web

# Build for production
bun run build

# Start production server
bun run start
```

## Environment Variables

```bash
# apps/web/.env
NEXT_PUBLIC_API_URL=http://localhost:8080
NEXT_PUBLIC_POSTHOG_KEY=your-posthog-key
NEXT_PUBLIC_POSTHOG_HOST=https://app.posthog.com
```

## Dependencies

Key dependencies:
- `next`: 16.0.6
- `react`: 19.2.0
- `@repo/common`: Common utilities
- `lucide-react`: Icon library
- `tailwindcss`: Styling

## Deployment

### Docker

```bash
# Build image
docker build -t tagaroa/web-app:latest -f apps/web/Dockerfile .

# Run container
docker run -p 3000:3000 tagaroa/web-app:latest
```

### Vercel

The app is optimized for deployment on Vercel:

```bash
# Deploy to Vercel
vercel deploy
```

## Customization

### Styling
The app uses Tailwind CSS 4. Customize styles in:
- Global styles: `src/app/globals.css`
- Tailwind config: `tailwind.config.ts`

### Content
Update marketing content in component files:
- Hero text: `src/components/hero-section.tsx`
- Navigation: `src/components/header-section.tsx`
- Footer: `src/components/footer-section.tsx`
